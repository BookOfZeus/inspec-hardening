
# Inspec

## Standalone Usage

This Compliance Profile requires [InSpec](https://github.com/chef/inspec) for execution:

```
$ git clone https://github.com/bookofzeus/inspec-harden-ubuntu
$ inspec exec inspec-harden-ubuntu
```
